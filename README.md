# competitive-contest-355

## Question: Huge Fibonacci Number Problem

Compute the n-th Fibonacci number modulo m.

![fib_img](sample.png)

- Input format. `Integers n and m.` 
- Output format. `Fn mod m`
-  Constraints. `1 ≤ n ≤ 1014` and `2 ≤ m ≤ 103`

Sample 1.

    Input:
        1 239
    Output:
        1

Sample 2.

    Input:
        115 1000
    Output:
        885

Sample 3.

    Input:
        2816213588 239
    Output:
        151
